URL
(Uniform Resource Locator): Es la dirección que se usa para acceder a un recurso en la web, como una página web, una imagen o un archivo. Contiene información sobre el protocolo, dominio, ruta y otros parámetros para localizar recursos específicos​( MDN Web Docs. )
HTTP
(Hypertext Transfer Protocol): Es un protocolo utilizado para la transferencia de información en la web. Permite la comunicación entre navegadores y servidores, facilitando la carga de páginas web y otros contenidos.
TCP/IP
(Transmission Control Protocol/Internet Protocol): Conjunto de protocolos que permiten la transmisión de datos en redes como Internet. TCP garantiza la correcta entrega de datos, mientras que IP asigna direcciones a los dispositivos para que puedan comunicarse entre sí​.
Internet
Es una red global que conecta millones de computadoras y dispositivos en todo el mundo, permitiendo el intercambio de información mediante diferentes protocolos, como TCP/IP.
W3C
(World Wide Web Consortium): Es una organización internacional que desarrolla estándares y directrices para garantizar el crecimiento a largo plazo de la web. Está involucrada en la creación de normas como HTML y CSS​.
HTML
(HyperText Markup Language): Es el lenguaje de marcado utilizado para crear páginas web. Define la estructura de los documentos web mediante etiquetas que describen diferentes elementos como texto, imágenes, enlaces, entre otros.
